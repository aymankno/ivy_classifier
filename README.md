# ivy_classifie

Binary image classifier that distinguishes my cat Ivy from other cats.

Built with ResNet18 transfer learning using PyTorch. Achieved around 98.63% test accuracy trained on ~1400 images of people, landscapes, and other cats.

1. Clone the repo
2. Set your dataset path in the notebook
3. Run all cells
4. Go to app.py and replace 'Ivy' with your pet's name!
5. Go to https://share.streamlit.io/new and log in with GitHub
6. Create the app and have fun! 😁
